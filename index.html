Machine Learning With Images


The task is to build the classifier for vehicles images recognizing using image descriptors. There is three classes of objects to recognize: personal transport, ambulance, police cars.

Methods

First of all, the data needed to be download with pickle. The features of each image were extracted. Then it was necessary to reduce its dimension. By using SVD, it is possible to get one dimension per feature. So a dataset of features has been done, one line per image. The number of features depends of the image. So, the empty columns has been filled with 0. At the end, some classifiers have been compared.
